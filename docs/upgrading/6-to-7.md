## Updates requirements

- Node v22
- Eleventy v3

## Update configuration options

- Plugin now uses a named export, `govukEleventyPlugin`
- Support for virtual templates means you can now [override the layouts](/layouts/#overriding-layouts) provided by the plugin
- Remove `dir.layouts`
- `navigation` renamed `serviceNavigation` and can accept all options for that component
- Can now show a search field in the navigation
- Deprecated `header.organisationName` option has been removed
- New `rebrand` option to enable rebranded GOV.UK header and footer components, icons and image assets

```diff
-
- import govukEleventyPlugin from '@x-govuk/govuk-eleventy-plugin'
+ import { govukEleventyPlugin } from '@x-govuk/govuk-eleventy-plugin'

  export default function(eleventyConfig) {
    // Register the plugin
    eleventyConfig.addPlugin(govukEleventyPlugin, {
-     navigation: {
-       items: [{
+     serviceNavigation: {
+       navigation: {
          text: 'API reference',
          href: '/api'
        }, {
          text: 'Glossary',
          href: '/glossary'
        }]
      },
+     rebrand: true
    })

    // Other configuration options
    ...

    dir: {
      input: 'docs',
-     layouts: '../node_modules/@x-govuk/govuk-eleventy-plugin/layouts'
    }
}
```
